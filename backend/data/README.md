## Local GIS Data

Keep raw GIS datasets local in this folder (shapefiles, DBF, PRJ, etc.).

- `backend/data/**` is gitignored to prevent oversized pushes.
- `backend/ph_muni.sql` is also gitignored.
- Only this `README.md` is tracked in git.

If data must be shared, use artifact storage or Git LFS instead of normal git blobs.
